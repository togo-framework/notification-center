# notification-center — docs

In-app notification inbox (database channel + bell) for togo.

- **Install:** `togo install togo-framework/notification-center`
- **Marketplace:** https://to-go.dev/plugins/notification-center
- **Source:** https://github.com/togo-framework/notification-center

See [usage.md](usage.md) for the Go API + REST endpoints. Complements the
`notifications` plugin (external delivery) by storing a per-user inbox.
